How to Take Screenshots:
-------------------------
1. Start both backend and frontend servers:
   - Terminal 1: cd server && node server.js
   - Terminal 2: cd client && npm start

2. Open http://localhost:3000 in your browser

3. Take screenshots using:
   - macOS: Cmd + Shift + 4 (select area to capture)
   - Save screenshots with the exact names listed above

4. Place all screenshots in this folder

5. The PROJECT_REPORT.md will automatically reference them
